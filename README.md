<!-- GETTING STARTED -->

### Run the webapp locally
1. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
2. Start the Flask server - navigate into repo directory and run - 
   ```sh
   python3 -m venv env
   source env/bin/activate
   pip install -r requirements.txt
   flask run
   ```
